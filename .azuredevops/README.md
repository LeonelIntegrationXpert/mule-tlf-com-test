# Azure DevOps

Este projeto consome templates do repositório global `release-flow-guardian-core`.

Crie um service connection chamado `github-service-connection` ou ajuste o campo `endpoint` nos YAMLs.

Variáveis necessárias no pipeline/variable group:

- ANYPOINT_CONNECTED_APP_CLIENT_ID
- ANYPOINT_CONNECTED_APP_CLIENT_SECRET
- ANYPOINT_ORG
- ANYPOINT_HOST
- EXCHANGE_GROUP_ID
